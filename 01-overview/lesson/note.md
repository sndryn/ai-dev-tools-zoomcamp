# Introduction

## Course Objective
To introduce how to use emerging AI tools to write code more efficiently

## AI Dev Tools

| Type | Characteristic | Example
| ---- | ---- | ---- |
| Chat Application |  Allow quick Q&A and prototyping,  require switching between the chat and IDE.| ChatGPT, Claude, DeepSeek, Ernie
| Coding Assistant | Embedded into editors/IDEs to provide inline code suggestions, test generation, multi-file edits, and more. | Claude Code, Gtihub Copilot, Cursor
| Project Bootstraper | Generate entire project from natural language prompts | Bolt, Lovable |
Agent | Autonomous coding helpers that combine LLMs with tools to read/write files, run commands, and manage projects | Antrophic computer use, PR Agent


## Glossary

**Vibe coding**: fully delegating coding to AI
